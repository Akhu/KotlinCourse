# Kotlin Course Samples

## Prequisities 
- Install Kotlin -> https://kotlinlang.org/docs/tutorials/command-line.html
- Install Gradle 

## Kotlin Basic Command Line
```
$ ./gradlew run
```

## Kotlin Gradle
```
$ ./gradlew run
```

## Kotlin JS
```
$ ./gradlew build
```
Then open in browser `index.html`
